# KLRfome Tutorial Notebooks

This directory contains tutorial notebooks for using KLRfome.

## Planned Notebooks

1. **01_quickstart.ipynb** - Basic usage with simulated data
2. **02_real_data.ipynb** - Working with real rasters and shapefiles
3. **03_hyperparameter_tuning.ipynb** - Cross-validation and tuning
4. **04_advanced_kernels.ipynb** - Customizing kernels
5. **05_section6_model_validation.ipynb** - Section 6 validation, sensitivities, and maps
6. **06_synthetic_methods_lab.ipynb** - Controlled M0--M3 methods laboratory

These notebooks will be created as part of the documentation phase.
